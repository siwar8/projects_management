# projects_management


Plateforme de gestion des projets au sein de l'entreprise

Cette application permet de gérer efficacement les projets, les clients et les employés au sein d'une entreprise. Elle propose une interface pour l’admin et une interface pour les employés, facilitant le suivi et la collaboration.

Fonctionnalités
Admin

Ajouter, modifier et supprimer des clients, employés et projets.

Créer des équipes (team) pour chaque projet.

Suivre l’avancement des projets avec des indicateurs clairs.

Employés

Accéder à la plateforme pour consulter la liste des projets et la liste des clients.

Ajouter des tâches pour les projets.

Suivre l’avancement des projets auxquels ils participent.

Technologies utilisées

Frontend : Angular

Backend : Node.js + Express

Base de données : MongoDB
